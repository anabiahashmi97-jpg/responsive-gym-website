# Gym24 - Fitness Website

Gym24 is a modern responsive gym landing page built using HTML, CSS, and JavaScript.
It features a clean dark UI, membership pricing cards, trainer showcase, and mobile responsive navigation.

## 🚀 Features

* Responsive Navbar with Hamburger Menu
* Hero Banner Section
* Reasons to Join Section
* Membership Pricing Cards
* About Us Section
* Trainers Showcase Grid
* Modern Footer Layout
* Fully Mobile Responsive

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Font Awesome
